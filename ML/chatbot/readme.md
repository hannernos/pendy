"수능 끝나자마자 동생은 바로 친구 만나러 나갔다. 나는 뭔가 잊은 거 같은데... 아! 학교!"와 같은 공부에 대한 관능적인 유머도 인기가 있을 것입니다. 즐겁게 웃을 수 있는 유머를 통해 어른들도 일상에서의 스트레스를 해소할 수 있도록 도움을 주시면 좋을 것 같습니다.
